# Cowat — Design Context

## Design Context

### Users
Grand public arrivant sur invitation (lien reçu par mail). Cibles : employés, entreprises, groupes souhaitant organiser le covoiturage en interne. L'inscription n'est pas ouverte librement — les intéressés doivent contacter par email. L'utilisateur type est un employé qui veut simplifier ses trajets quotidiens et réduire son empreinte carbone.

### Brand Personality
Friendly, pratique, moderne. L'interface doit inspirer confiance et légèreté — pas corporate, pas startup-cool. Quelque chose de chaleureux et humain, avec une pointe de caractère (la vache, le nom Cowat, l'aspect communautaire).

### Aesthetic Direction
- Ambiance : organique/naturelle teintée de modernité — verts, terres, chaleur (pas cyan-sur-noir, pas gradient purple)
- Thèmes : écologie, communauté, trajet quotidien, simplicité
- Light mode par défaut, dark mode supporté
- Typographie : Outfit Variable (déjà en place) — utiliser les graisses pour créer de la hiérarchie
- Anti-références : landing pages SaaS génériques avec hero métrique + gradient, glassmorphisme, icônes arrondies identiques

### Design Principles
1. **Chaleureux avant tout** — chaque écran doit sembler fait pour des humains, pas pour un deck investisseur
2. **Sobriété intentionnelle** — peu d'éléments, mais chacun compte ; pas de remplissage
3. **Éco-cohérence** — les choix visuels (couleurs, espacement, typographie) évoquent naturellement la durabilité
4. **Communauté visible** — suggérer qu'on rejoint quelque chose, pas qu'on souscrit à un produit
5. **Mobile-first réel** — les trajets se vivent sur mobile ; la landing doit être parfaite sur petit écran
